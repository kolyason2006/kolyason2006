## 📂 Featured Projects

### [gRPC Mock Server](https://github.com/kolyason2006/grpc-demo)
![Go](https://img.shields.io/badge/Go-00ADD8?style=flat-square&logo=go&logoColor=white) ![gRPC](https://img.shields.io/badge/gRPC-4285F4?style=flat-square&logo=google&logoColor=white) ![Redis](https://img.shields.io/badge/Redis-DC382D?style=flat-square&logo=redis&logoColor=white)

A high-performance utility designed for the dynamic mocking of gRPC services. Its core technical feature is the ability to parse any .proto file using protoreflect without the need for code generation. This enables the instant deployment of isolated gRPC servers with fully customizable responses for any service contract directly at runtime.

### [API Gateway](https://github.com/kolyason2006/api-gateway)
![C++](https://img.shields.io/badge/C++-00599C?style=flat-square&logo=c%2B%2B&logoColor=white) ![Boost](https://img.shields.io/badge/Boost.Asio-00599C?style=flat-square&logo=c%2B%2B&logoColor=white) ![Redis](https://img.shields.io/badge/Redis-DC382D?style=flat-square&logo=redis&logoColor=white)

A high-performance API gateway engine architected for low-latency request routing and high-throughput processing. Its core technical feature is a fully asynchronous, non-blocking I/O model leveraging Boost.Asio to minimize context-switching overhead and maximize CPU efficiency. This enables the deployment of scalable gateways with integrated rate limiting and distributed state management, powered by Redis for consistent performance across high-load environments.
